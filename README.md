✊✋✌️ Jokenpô em Ruby

Este é um projeto de console desenvolvido em Ruby para colocar em prática fundamentos da linguagem, como métodos, estruturas de repetição, condicionais e manipulação de entrada de dados.

🚀 Funcionalidades

Interação via terminal com o usuário.

Lógica de decisão baseada nas regras clássicas do Jokenpô.

Oponente virtual com escolhas aleatórias.

Opção de reiniciar a partida sem fechar o programa.

🛠️ Tecnologias Utilizadas

Linguagem: Ruby 3.x

Conceitos Aplicados: Métodos (def), Lógica Condicional (if/elsif), Captura de Input (gets) e Randomização (rand).

🎮 Como Jogar?

Início: Ao executar o script, insira seu nome para ser saudado pelo sistema.

Escolha: Selecione sua jogada digitando o número correspondente:

1 - Pedra

2 - Papel

3 - Tesoura

Resultado: O computador fará a escolha dele e o sistema exibirá quem venceu a rodada.

Replay: Ao final, você pode digitar s para jogar novamente ou qualquer outra tecla para encerrar.

🔧 Execução

Para rodar o jogo localmente, certifique-se de ter o Ruby instalado e execute:

ruby jokenpo.rb
